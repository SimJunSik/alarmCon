# alarmCon

안드로이드 알림을 감지해 **앱별/발신자별로 커스텀 진동 패턴**을 적용하는 앱입니다.

## 주요 기능

- **앱별 패턴**: 특정 앱의 알림에 지정한 진동 패턴 적용
- **발신자/키워드 패턴**: 메시지 앱에서 특정 발신자(또는 키워드)일 때 별도 패턴 적용
- **우선순위 규칙**: 발신자 패턴이 있으면 그게 우선, 없으면 앱 기본 패턴 적용
- **무음 옵션**: 발신자 미일치 시 무음 처리 가능
- **패턴 관리**: 패턴 이름 지정/목록 확인/삭제
- **패턴 미리보기**: 테스트 진동 버튼으로 즉시 확인
- **사용법 페이지**: 앱 내 도움말 화면 제공

## 동작 원리

1. `NotificationListenerService`로 다른 앱 알림을 감지
2. 알림 제목/대화 정보에서 발신자 텍스트 추출
3. 발신자 패턴 → 앱 기본 패턴 순으로 매칭
4. 일치하는 패턴이 있으면 진동 실행

## 사용 방법

1. 앱 실행 후 **알림 접근 권한** 허용
2. 앱 검색 → 원하는 앱 선택
3. (선택) 발신자/키워드 입력
4. 패턴 이름과 진동 패턴 저장
5. 필요 시 **무음 토글** 활성화

## 주의 사항

- 발신자 추출은 **앱/OS 설정에 따라 제한**될 수 있습니다.
- 대상 앱의 알림이 꺼져 있으면 감지가 불가합니다.
- 제조사/안드로이드 버전에 따라 알림 내용이 마스킹될 수 있습니다.
